# Netflix Content Analysis

This project explores trends in Netflix movies and TV shows using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Movies vs TV Shows distribution
- Netflix content growth over time
- Top countries producing content
- Genre distribution
- Content rating analysis
- Release year trends

## Key Insights
- Netflix has significantly more movies than TV shows.
- The platform experienced major growth around 2018–2019.
- International movies and dramas dominate the catalog.
- Most Netflix content targets teen and adult audiences.
